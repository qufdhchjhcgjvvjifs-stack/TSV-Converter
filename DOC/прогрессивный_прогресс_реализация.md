## 📋 ОПИСАНИЕ

Реализована расширенная система отображения прогресса конвертации с детальными метриками производительности.

### Что было добавлено:

1. **Класс `ProgressData`** — структура данных для хранения метрик прогресса
2. **Класс `ProgressTracker`** — трекер для расчёта ETA, скорости и времени
3. **Новый сигнал `progress_data`** — передача детальных данных в GUI
4. **Панель детального прогресса** — UI компонент с 6 метриками
5. **Методы обновления и сброса** — актуализация информации в реальном времени

### 1. `converter.py`

**Новые классы:**

```python
@dataclass
class ProgressData:
    """Структура данных для метрик прогресса."""
    percent: int              # Процент (0-100)
    processed_rows: int       # Обработано строк
    total_rows: int           # Всего строк
    elapsed_seconds: float    # Прошло времени
    eta_seconds: float        # Осталось времени (ETA)
    rows_per_second: float    # Скорость
    current_file: str         # Имя файла
    current_operation: str    # Текущая операция
    
    # Методы форматирования:
    def format_eta() -> str        # "02:15" или "01:30:45"
    def format_elapsed() -> str    # "00:45"
    def format_speed() -> str      # "480 строк/сек" или "1.2K строк/сек"


class ProgressTracker:
    """Трекер прогресса с расчётом метрик."""
    
    def __init__(self, total_rows: int = 0)
    def start()                    # Запуск таймера
    def update(rows) -> ProgressData  # Обновление и возврат метрик
    def reset(new_total: int = 0)  # Сброс для новой задачи
```

**Изменения в `TSVToExcelConverter`:**

```python
class TSVToExcelConverter(QThread):
    # Новый сигнал
    progress_data = Signal(object)  # ProgressData
    
    def __init__(self, ...):
        self.progress_tracker = ProgressTracker()
    
    def run(self):
        self.progress_tracker.reset(self.total_rows)
        self.progress_tracker.start()
        
        # В цикле конвертации:
        progress_data = self.progress_tracker.update(row_num)
        if progress_data:
            progress_data.current_file = os.path.basename(tsv_file)
            progress_data.current_operation = "Запись данных..."
            self.progress_data.emit(progress_data)
```

### 2. `gui.py`

**Новые компоненты в `MainWindow`:**

```python
class MainWindow(QMainWindow):
    def __init__(self):
        # Для обновления прогресса
        self._progress_timer = QTimer()
        self._current_progress_data = None
    
    def _create_control_panel(self):
        # Панель детального прогресса (6 метрик)
        self.progress_details_widget = QWidget()
        self.progress_file_label = QLabel()      # 📁 Файл
        self.progress_operation_label = QLabel() # 🔄 Операция
        self.progress_rows_label = QLabel()      # 📊 Обработано
        self.progress_elapsed_label = QLabel()   # ⏱ Прошло
        self.progress_eta_label = QLabel()       # ⏳ Осталось
        self.progress_speed_label = QLabel()     # ⚡ Скорость
    
    def update_progress_details(self, progress_data):
        """Обновляет метрики из ProgressData."""
        self.progress_details_widget.setVisible(True)
        self.progress_file_label.setText(progress_data.current_file)
        self.progress_operation_label.setText(progress_data.current_operation)
        self.progress_rows_label.setText(
            f"{progress_data.processed_rows:,} / {progress_data.total_rows:,}"
        )
        self.progress_elapsed_label.setText(progress_data.format_elapsed())
        self.progress_eta_label.setText(progress_data.format_eta())
        self.progress_speed_label.setText(progress_data.format_speed())
    
    def reset_progress_details(self):
        """Скрывает панель и сбрасывает метрики."""
```

### 3. `main.py`

**Подключение сигнала:**

```python
class TSVConverterApp:
    def _start_conversion(self):
        # Подключение нового сигнала
        self.converter.progress_data.connect(
            self.window.update_progress_details
        )
    
    def _on_conversion_finished(self):
        # Сброс панели после завершения
        self.window.reset_progress_details()
    
    def _on_conversion_error(self, error):
        # Сброс панели при ошибке
        self.window.reset_progress_details()
```

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Расчёт метрик:

**1. Процент выполнения:**
```python
percent = int((processed_rows / total_rows) * 100)
```

**2. Скорость обработки:**
```python
speed = processed_rows / elapsed_time
```

**3. ETA (оставшееся время):**
```python
remaining_rows = total_rows - processed_rows
eta = remaining_rows / speed
```

**4. Форматирование времени:**
```python
# Если часов нет: "MM:SS"
# Если часы есть: "HH:MM:SS"
```

### Оптимизация:

**Защита от слишком частых обновлений:**
```python
class ProgressTracker:
    min_update_interval = 0.1  # сек
    
    def update(self):
        if current_time - last_update < 0.1:
            return None  # Пропускаем обновление
```

Это предотвращает:
- Перегрузку UI обновлениями
- Избыточную отправку сигналов
- Мерцание интерфейса
