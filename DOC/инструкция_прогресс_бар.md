# 📈 Прогрессивный Прогресс-Бар — Инструкция Пользователя

## Что изменилось?

Теперь во время конвертации вы видите **детальную информацию** о прогрессе, а не просто процент выполнения.

---

## 🎯 Как это выглядит?

### Раньше:
```
████████████░░░░░░░░ 65%
Время: 02:15
```

### Теперь:
```
████████████░░░░░░░░ 65%
Время: 02:15

┌─────────────────────────────────────────────────────────────┐
│  📁 Файл: data_2024.tsv          🔄 Операция: Запись данных │
│  📊 Обработано: 65,000 / 100,000 ⏱ Прошло: 00:02:15        │
│  ⏳ Осталось: 00:01:30           ⚡ Скорость: 480 строк/сек │
└─────────────────────────────────────────────────────────────┘
```

---

## 📊 Какие метрики отображаются?

| Метрика | Описание | Пример |
|---------|----------|--------|
| **📁 Файл** | Имя текущего обрабатываемого файла | `data_2024.tsv` |
| **🔄 Операция** | Что происходит прямо сейчас | `Чтение файла...`, `Запись данных...`, `Создание сводной...` |
| **📊 Обработано** | Сколько строк обработано из скольких | `65,000 / 100,000` |
| **⏱ Прошло** | Сколько времени уже прошло | `00:02:15` |
| **⏳ Осталось (ETA)** | Сколько ещё ждать (прогноз) | `00:01:30` |
| **⚡ Скорость** | Производительность конвертации | `480 строк/сек` |

---

## 🔍 Когда это полезно?

### ✅ Большие файлы (100K+ строк)
Вы видите:
- Сколько ещё ждать (ETA)
- С какой скоростью идёт обработка
- На каком этапе возникла проблема (если что-то пошло не так)

### ✅ Пакетная обработка нескольких файлов
Вы видите:
- Какой файл сейчас обрабатывается
- Общий прогресс по всем файлам

### ✅ Долгие операции (сводные таблицы, разделение)
Вы видите:
- Текущую операцию
- Прогноз времени завершения

---

## 💡 Как интерпретировать метрики?

### Скорость обработки:
- **< 100 строк/сек** — Медленно (возможно, сложный файл или сетевой диск)
- **100-500 строк/сек** — Нормальная скорость
- **500-1000 строк/сек** — Хорошая скорость
- **> 1000 строк/сек** — Отлично (1K+ строк/сек)

### ETA (оставшееся время):
- **Меняется сильно** — Скорость нестабильна, прогноз неточный
- **Стабильно уменьшается** — Всё идёт по плану
- **Растёт** — Скорость упала (возможно, файл больше ожидаемого)

### Формат времени:
- **MM:SS** — Минуты:Секунды (если меньше часа)
- **HH:MM:SS** — Часы:Минуты:Секунды (если больше часа)

## ⚙️ Технические детали

### Как это работает?

1. **Трекер ProgressTracker** вычисляет метрики в реальном времени
2. **Сигнал progress_data** передаёт данные из потока конвертации в GUI
3. **Панель обновляется** каждые 0.1 секунды (10 раз в секунду)

### Почему панель скрыта после завершения?

- Не загромождает интерфейс
- Вы уже видите результат (файл создан)
- Можно сосредоточиться на следующих действиях

## ❓ Частые вопросы

### Почему ETA меняется?
ETA рассчитывается на основе текущей скорости. Если скорость меняется (например, файл неоднородный), ETA будет корректироваться.

### Почему скорость иногда падает?
Возможные причины:
- Файл имеет сложную структуру
- Сетевой диск (медленный доступ)
- Антивирус проверяет файл
- Другие программы нагружают диск

### Можно отключить панель?
В текущей версии — нет. Но она не мешает работе и автоматически скрывается после завершения.

### Почему метрики обновляются не каждую секунду?
Обновление 10 раз в секунду обеспечивает:
- Плавность отображения
- Отсутствие мерцания
- Минимальную нагрузку на систему

## 📝 Примеры использования

### Пример 1: Конвертация большого файла

```
Файл: sales_2024.tsv (500,000 строк)

Начало:
  📊 Обработано: 0 / 500,000
  ⏱ Прошло: 00:00
  ⏳ Осталось: --:--
  ⚡ Скорость: 0 строк/сек

Через 1 минуту:
  📊 Обработано: 45,000 / 500,000
  ⏱ Прошло: 00:01:15
  ⏳ Осталось: 00:09:45
  ⚡ Скорость: 600 строк/сек

Через 5 минут:
  📊 Обработано: 250,000 / 500,000 (50%)
  ⏱ Прошло: 00:05:30
  ⏳ Осталось: 00:05:30
  ⚡ Скорость: 757 строк/сек

Завершение:
  📊 Обработано: 500,000 / 500,000 (100%)
  ⏱ Прошло: 00:11:00
  ⏳ Осталось: 00:00:00
  ⚡ Скорость: 757 строк/сек
```

### Пример 2: Пакетная обработка

```
Файлы: jan.tsv, feb.tsv, mar.tsv (по 100K строк каждый)

Общий прогресс:
  📁 Файл: feb.tsv
  📊 Обработано: 150,000 / 300,000
  
  jan.tsv ✅ завершён
  feb.tsv 🔄 обрабатывается (50%)
  mar.tsv ⏳ в очереди
```

## 🚀 Советы по использованию

1. **Смотрите на ETA** для планирования времени
2. **Следите за скоростью** — резкое падение может указывать на проблему
3. **Проверяйте имя файла** при пакетной обработке
4. **Обращайте внимание на операцию** — если застряло на одном этапе, возможно, есть проблема
